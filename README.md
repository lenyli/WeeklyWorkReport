# 智慧云影研发部工作周报

这是一个用于收集工作周报的小工具。打开网页后输入姓名即可填写周报；提交后，内容会自动保存到统一收集端，并可导出为 Excel。

## 启动

在项目目录运行：

```powershell
node server.js
```

默认访问地址：

```text
http://127.0.0.1:5173/
```

如果需要指定端口：

```powershell
$env:PORT="5174"; node server.js
```

然后访问：

```text
http://127.0.0.1:5174/
```

## 局域网访问

手机或其他电脑访问时，把 `127.0.0.1` 换成运行服务电脑的局域网 IP 地址，例如：

```text
http://192.168.1.20:5174/
```

运行服务的电脑需要和访问设备在同一个网络中。

## 使用

1. 打开网页。
2. 输入姓名进入。
3. 选择本周周一日期。
4. 按日期填写：
   - 开发/测试/需求
   - 场地支持
   - 售前支持
   - 其他
5. 点击“提交”。

草稿会保存在当前设备浏览器中；提交后的数据会保存到服务端。

## 导出 Excel

填写页面点击“下载个人 Excel”可下载当前人员的周报 Excel。

在汇总页面选择周一日期后，点击“导出 Excel”即可生成汇总文件。

导出的 Excel 包含：

- 汇总表
- 每个人一个 sheet
- 未提交状态
- 成员 sheet 按姓名拼音排序

## 安装为 PWA

在支持的浏览器中打开网页后，可以选择“安装应用”或“添加到主屏幕”。

安装后可以像普通应用一样从桌面或主屏幕打开。

## 数据位置

服务端数据保存在：

```text
data/store.json
```

如需重新开始收集，可在停止服务后删除该文件。下次启动并使用时会自动重新创建。

## 文件说明

- `server.js`：本地服务和数据保存
- `index.html`：页面结构
- `app.js`：页面交互和 Excel 导出
- `styles.css`：页面样式
- `manifest.webmanifest`：PWA 配置
- `sw.js`：离线缓存
- `icons/`：应用图标
