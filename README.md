# 智慧云影研发部工作周报

这是一个纯静态 PWA 周报工具，打开网页即可填写、保存草稿、分享或下载个人 Excel。汇总时可上传成员发来的个人 Excel，并在浏览器本地生成汇总 Excel。

## 打开和安装

把项目部署到任意静态网页托管后，使用浏览器打开页面即可。

在手机或电脑支持 PWA 的浏览器中，可以选择“安装应用”或“添加到主屏幕”。安装后可像普通应用一样从桌面或主屏幕打开。

## 填写周报

1. 打开页面。
2. 在登录页填入名字即可进入。
3. 周一日期会自动更新为本周周一，也可以手动选择其他周。
4. 填写周一到周五的工作内容。
5. 点击“保存草稿”会保存到当前设备的浏览器。
6. 点击“清空”只会清空当前表格内容。
7. 页面底部可以“分享周报”或“下载个人 Excel”。

手机端如果浏览器支持文件分享，会弹出系统分享面板，可选择微信、邮件或其他应用发送。电脑端如果不支持文件分享，直接下载个人 Excel 即可。

## 汇总周报

1. 打开页面并填入指定人姓名进入。
2. 选择要汇总的周一日期。
3. 点击“上传成员 Excel”，选择成员发来的个人周报文件。
4. 页面会显示已提交和未提交人员。
5. 成员名单会根据上传文件自动增加，也可以手动添加或删除。
6. 点击“导出 Excel”生成汇总文件。

汇总文件中每个人一个 sheet，并按姓名首字母排序。

## 本地数据

草稿、已上传名单和已解析的成员周报会保存在当前设备的浏览器中。

如果更换设备、清理浏览器数据或更换浏览器，需要重新填写或重新上传成员 Excel。

## 使用说明

页面内有“使用说明”按钮，点击后会弹出每个功能的说明。
