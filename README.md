# 智慧云影研发部工作周报

这是一个纯静态 PWA 工作周报工具。成员填写后生成个人 Excel，可下载或通过系统分享发送；汇总人员上传成员 Excel 后，在浏览器本地生成汇总 Excel。

## 打开方式

部署到任意静态网页托管即可，例如 GitHub Pages。

也可以直接在本机用静态服务器预览：

```bash
python3 -m http.server 5174
```

然后打开：

```text
http://127.0.0.1:5174/
```

## 成员填写

1. 打开 PWA 页面。
2. 输入姓名进入。
3. 选择本周周一日期。
4. 填写周一到周五的内容。
5. 点击“保存草稿”可保存到当前设备浏览器。
6. 点击“下载个人 Excel”下载自己的周报文件。
7. 手机或支持文件分享的浏览器会显示“分享周报”，点击后可通过系统分享发送给汇总人员。

电脑端如果浏览器不支持文件分享，只需要下载个人 Excel 后自行发送。

## 汇总人员操作

1. 打开 PWA 页面。
2. 输入汇总人员姓名进入。
3. 选择要汇总的周一日期。
4. 点击“上传成员 Excel”，选择成员发来的个人周报文件。
5. 页面会显示已提交和未提交情况。
6. 点击“导出 Excel”生成汇总文件。

可以一次选择多个成员 Excel。汇总文件中每个人一个 sheet，并按姓名拼音排序。

## 本地数据

草稿、已上传名单和已解析的成员周报会保存在当前设备浏览器中。

如果换设备或清理浏览器数据，需要重新上传成员 Excel。

## 安装为 PWA

在支持的浏览器中打开网页后，可以选择“安装应用”或“添加到主屏幕”。

安装后可以像普通应用一样从桌面或主屏幕打开。

## 文件说明

- `index.html`：页面结构
- `app.js`：填写、分享、上传解析和 Excel 导出
- `styles.css`：页面样式
- `manifest.webmanifest`：PWA 配置
- `sw.js`：静态缓存
- `icons/`：应用图标
