# 智慧云影研发部工作周报

这是一个使用浏览器本地数据库占位的 PWA 周报工具。成员可以填写、保存草稿并提交周报；leader 可以查看账号、提交情况并生成汇总 Excel。

当前版本用于验证业务流程，账号和提交数据保存在当前浏览器的 IndexedDB 中，不同设备之间暂时不会同步。后续确定云端部署方案后，可将 `db.js` 替换为云端 API 数据层。

部署到 Cloudflare Pages 后，可以在家里和公司电脑上打开同一个网址，但两台电脑仍分别使用各自的本地数据库。

## 打开和安装

把项目部署到任意静态网页托管后，使用浏览器打开页面即可。

在手机或电脑支持 PWA 的浏览器中，可以选择“安装应用”或“添加到主屏幕”。安装后可像普通应用一样从桌面或主屏幕打开。

## 填写周报

1. 打开页面。
2. 使用姓名和六位数字密码登录。
3. 初始 leader 账号为“周南”，初始密码为 `123456`。
4. 新成员第一次使用姓名和 `123456` 登录时，会自动创建普通成员账号。
5. 周一日期会自动更新为本周周一，也可以手动选择其他周。
6. 填写周一到周五的工作内容，点击“提交周报”写入本地数据库。
7. 再次提交会覆盖本人同一周的已提交版本。
8. 页面右上角可修改密码；密码必须正好是六位数字。

手机端如果浏览器支持文件分享，会弹出系统分享面板，可选择微信、邮件或其他应用发送。电脑端如果不支持文件分享，直接下载个人 Excel 即可。

## 汇总周报

1. 使用 leader 账号登录。
2. 选择要汇总的周一日期。
3. 在成员名单中删除已经登录过的成员账号；删除账号不会删除历史周报。
4. 页面会按照现有账号和保留的历史周报显示已提交和未提交人员。
5. 仍可上传旧版成员 Excel，作为兼容的补录方式。
6. 点击“导出 Excel”生成汇总文件。

汇总文件中每个人一个 sheet，并按姓名首字母排序。

## 本地数据

草稿保存在 localStorage；账号和正式提交保存在 IndexedDB；旧版上传文件的解析结果保存在 localStorage。

如果更换设备、清理浏览器数据或更换浏览器，需要重新填写或重新上传成员 Excel。

## 当前限制与云端迁移

- 当前版本不是多人共享数据库，同一台设备上的账号用于模拟完整流程。
- Windows、macOS、Android 和 iOS 之间的同步，需要部署云端 API、数据库和服务端身份校验。
- 云端版可继续保留现在的 PWA 页面与离线草稿，联网后由数据层负责同步。
- 正式上线时必须在服务器端使用专用密码哈希算法和随机盐，不能沿用本地占位实现。
